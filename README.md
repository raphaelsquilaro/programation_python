# 🐍 Curso de Programação em Python

Repositório criado para armazenar meus estudos, exercícios e projetos desenvolvidos durante o curso de Programação em Python.

O objetivo deste repositório é acompanhar minha evolução na linguagem, praticando desde conceitos básicos até aplicações com interface gráfica, APIs e automações.

---

# 📚 Conteúdo do Curso

## ✅ Fundamentos do Python
- Variáveis
- Tipos de dados
- Operadores
- Entrada e saída de dados
- Conversões

## 🔁 Estruturas de Controle
- Condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`for`, `while`)
- Match Case

## 📦 Estruturas de Dados
- Listas
- Tuplas
- Dicionários
- Sets

## ⚙️ Funções
- Criação de funções
- Parâmetros
- Retorno
- Funções lambda

## 🧠 Conceitos Intermediários
- Tratamento de erros
- Manipulação de arquivos
- Módulos e pacotes
- Programação Orientada a Objetos (POO)

## 🖥 Projetos Desenvolvidos
- Calculadora de IMC
- Consulta de CNPJ com API
- YouTube Downloader
- Interfaces gráficas com Tkinter e CustomTkinter
- Projetos em console
